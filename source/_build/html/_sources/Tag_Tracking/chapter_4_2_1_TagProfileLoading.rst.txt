.. _4.2.1 Tag Profile Loading:

TagProfileLoading
===============================


描述
---------------------------

TagProfileLoading主要实现对需要识别的定位信标的标定文件的加载，我们需要根据实际使用的Tag类型来勾选不同的参数，
将鼠标移至参数名称上可查看其支持的ID范围，具体配置如下图。

.. Attention::
    一个场景里面有且只能有一个此组件，超过1个时，识别Tag会出现闪退

.. image:: /_static/tagprofileloading.png



通常我们的场景中都需要识别Beacon，所以一般情况下我们创建的TagProfileLoading都需要勾选Beacon。

如果您需要对其他类型的定位信标进行识别，那么您可以对照以下表格属性进行勾选

============================     =====================================================================================================================
属性                                  描述
============================     =====================================================================================================================
Beacon                             启用加载Tag类型为Beacon，ID为65-67的标定参数。Beacon1，2，3编号分别对应ID65，66，67
LiBeacon                           启用加载Tag类型为LiBeacon的标定参数，勾选后推荐选择Li Beacon Type Everything，如有特殊要求，可单独选择某个ID。
TopoTag                            启用加载Tag类型为TopoTag的标定参数，可根据实际使用的定位信标尺寸选择Topo Tag_450mm或者Topo Tag_350mm。
SingleCard                         启用加载Tag类型为SingleCard，ID为18，19，22，23，25，39的标定参数。
Gun                                启用加载Tag类型为Gun的标定参数，勾选后推荐选择Guntype Everything，如有特殊要求，可选择其他类型
============================     =====================================================================================================================

    LiBeacon类型与gunsight类型中ID冲突，所以无法同时使用。


使用方法
-----------------------------

1. 在Hierarchy界面点击鼠标右键选择Ximmerse XR->Tag Profile Loading

.. image:: /_static/tagprofileloadingcreate.png

2. 根据您需要使用的定位信标勾选对应的标定参数加载选项

.. image:: /_static/tagprofileloading.png
