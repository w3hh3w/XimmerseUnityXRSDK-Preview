4.SDK功能介绍

此章节主要介绍与RhinoX Pro硬件特性相关的一些功能模块，如手柄输入，手势识别，Tag跟踪等。具体Unity XR相关的使用方法 请查看 Unity官方文档 。

4.1 手柄&头显输入

除了XR Interatcion Toolkit提供的常规交互功能之外，如果您需要使用手柄或者头显来实现额外的定制化功能，如通过按键 调出或者隐藏UI，触碰到物体后手柄振动等，您可以参考此章节类容进行开发

Note

XR Interatcion Toolkit里面自带的组件已经实现了常规的手柄交互功能，如果您只是需要实现抓握键抓取 物体，扳机键触发UI等常规交互功能，直接使用自带组件即可，不需要使用此处的功能。

详细信息请查看如下章节

  • 手柄按键/振动
  • 头显按键

4.2 Tag跟踪

Tag跟踪是我们自研的一种定位技术，主要用于虚拟场景与现实场景的空间锚定，以及实现对特殊工具的物体跟踪。

详细信息请查看如下章节：

  • TagProfileLoading
  • Ground Plane
  • Tracking Target
  • CreatesGroundPlaneByJson
  • Tracking Event
  • Ground Fusion Event

4.3 手部跟踪

手部跟踪是燧光最新推出的一种交互方式，该功能会跟踪并识别手部姿势，并会根据特定姿势触发交互功能。

Note

此功能处于测试阶段，后续将会对其功能进行优化，目前仅支持RhinoX Pro设备。
